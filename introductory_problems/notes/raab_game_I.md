### 題目意思
* 有兩個玩家 A、B
* 每人都有 n 張牌，數字為 1 ~ n，各用一次
* 共進行 n 回合
* 每回合：
  * A、B 各出一張牌
  * 數字大的得 1 分
  * 相同則 平手（0 分）
* 已知最後分數為 (A = a, B = b)
* 問：
  1. 是否存在一種出牌順序
  2. 若存在，輸出其中一種

### 核心概念
* 因為每回合最多得 1 分，且總共 n 回合 &rarr; `a + b <= n`
* 不合理的情況：
  * `a + b > n`
  * `a == 0 && b > 0`、`a > 0 && b == 0`
    > 若一方在所有回合皆未贏（$A_i \leq B_i$），
      而另一方至少贏一回合（$\exist j$：$A_j < B_j$），
      則總和必不相等，
      &rarr; 與 $\displaystyle \sum_{i = 1}^n A_i = \displaystyle \sum_{i = 1}^n B_i$ 事實矛盾
* 合理情況：`a == 0 && b == 0`、`a > 0 && b > 0`
* 構造策略：分成三種情況
  | 類型 | 次數 |
  |:---:|:----:|
  |平手 | `k = n - a - b` |
  |B 贏 | `b` |
  |A 贏 | `a` |
* 數字區間：
    A 的牌：
    | 區間 | 用途 |
    |:------:|:---:|
    | `1 .. k`     | 平手 |
    | `k+1 .. k+b` |  A 輸（B 贏）|
    | `b+k+1 .. n` | A 贏 |

    B 的牌：
    | 區間           | 用途       |
    | :----------: | :------: |
    | `1 .. k`     | 平手 |
    | `k+1 .. k+a` | B 輸（A 贏）|
    | `k+a+1 .. n` | B 贏      |

### 問題點
* 合理情況和不合理的條件想不到
* A 牌和 B 牌的區間範圍也想不到
