### 題目意思
* 題目給一個 `n × n` 的西洋棋棋盤，上面只有一種棋子：騎士
* 計算棋盤上的「每一個格子 `(i, j)`」，最少需要幾步才能走到左上角 `(0, 0)`

### 核心概念
* 可以轉換成圖論問題：無權圖最短路徑問題
  * 每一個格子 `(i, j)` 是一個「節點」
  * 騎士能走到的格子，就是這個節點的「鄰居」
  * 每條邊的權重都是 `1`
* 使用 BFS，因為 BFS 可以用來解**每一步代價相同的最短路徑問題**
* 每一格 → `(0, 0)` 的最短距離 &rArr; 從 `(0, 0)` 出發 → 走到所有格子的最短距離
* 對每個合法、尚未拜訪的格子 `(nx,ny)`：
  * `dist[nx][ny] = dist[x][y] + 1`
  * 加入 queue
* 騎士有 8 種移動方式：
    ```cpp
    int dx[8] = { 2, 2, -2, -2, 1, 1, -1, -1 };
    int dy[8] = { 1, -1, 1, -1, 2, -2, 2, -2 };
    ```
### 問題點
