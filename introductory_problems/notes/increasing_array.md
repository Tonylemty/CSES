### 題目意思
1. 將原本的陣列變為**非遞減陣列**，也就是 `x[1] ≤ x[2] ≤ ... ≤ x[n]`
2. 唯一操作：選一個元素，將它 +1
3. 問：最少需要幾次

### 核心概念：Greedy
從左到右掃過整個陣列：
* 對每個位置 `i`，如果 `x[i] < x[i - 1]`
  &rarr; 將 `x[i]` 增加到至少 `x[i - 1]`
  &rarr; 增加次數是 `x[i - 1] - x[i]`

* 如果是 `x[i] >= x[i - 1]`
  &rarr; 不需要增加

* 使用 Greedy 的原因
因為每個位置只需要補到「剛好合法」的最低值，補多了浪費、補少了不合法。每次都補到最低合法值 → 最少 moves → 這就是 Greedy
