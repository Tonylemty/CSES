### 題目意思
* 給定一個長度為 $n$ 的正整數陣列，要求將陣列分成 $k$ 段連續子陣列
* 目標：讓 $k$ 段中「最大總和」最小，輸出這個最小值

### 核心概念
* 二分搜尋答案：
  * 下界：陣列最大值（每段至少要容納最大元素）
  * 上界：陣列總和（全部放同一段）
* 貪心檢查：
  * 猜一個上限 $mid$，用貪心方式分段
  * 每段累加，超過 $mid$ 就開新段
  * 計算總共需要幾段，若 $\le k$ 就合法
* 單調性：
  * 若 $mid$ 合法，$mid+1$ 也一定合法
  * 若 $mid$ 合法，$mid-1$ 不一定合法，因為上限越小，分段越多，越難達到 $\le k$ 段
  * 用二分搜尋找最小合法 $mid$
  
* 二分搜尋主流寫法：
  * 本質相同，都是利用「單調性」排除一半的不可能：
  * 縮圈法：while (l < r) 搭配 r = mid 與 l = mid + 1。結束時 l 即為答案。
  * 存檔法（最推薦）：while (l <= r) 搭配 ans = mid; r = mid - 1 與 l = mid + 1。
  * 時間複雜度：$O(n \log S)$，$S$ 為陣列總和
