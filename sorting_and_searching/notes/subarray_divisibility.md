### 題目意思
* 在一個包含 $n$ 個整數的陣列中，找出有多少個「連續子陣列（Subarray）」，其內部的數字總和可以被 $n$ 整除

### 核心概念
* 子陣列與前綴和的轉換：
  任何連續子陣列的和，都可以看作是兩個「前綴和」的差。
  $$ \text{SubarraySum}(i, j) = S[j] - S[i - 1]$$
  * $S[i]$：從開頭到目前的總和
  * $S[i-1]$：之前某個點的總和
* 同餘定理：
  * 題目要求 $\text{SubarraySum} \pmod n = 0$，代入公式：$$(S[j] - S[i-1]) \pmod n = 0 \implies S[j] \pmod n = S[i-1] \pmod n$$
  * 結論：只要兩個前綴和的「餘數」相同，它們夾出來的區間和就一定能被 $n$ 整除
* 負數餘數的修正：
  * 負數取模會得到負餘數。為了統一統計，必須將其轉為正餘數：
    ```c++
    remainder = (current_sum % n + n) % n;
    ```
    這能確保所有餘數都落在 $0$ 到 $n-1$ 之間
* 統計與紀錄：
  * 只需要記住：「目前為止，每一種餘數分別出現過幾次？」
  * 使用一個陣列 `cnt[n]` 來記錄
  * 初始條件 `cnt[0] = 1`：這代表「空陣列」的餘數是 0，因為 `0 % 0 = 0`，用來匹配那些「從頭開始就剛好整除」的子陣列
